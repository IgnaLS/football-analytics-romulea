# Football Performance Analytics - Romulea Season 2025/26

## Overview
Comprehensive tactical and statistical analysis of Romulea's home matches in Eccellenza Lazio (Italian fifth-tier championship), analyzing offensive and defensive patterns through data visualization and metrics.

## Methodology
- **Data Collection**: Manual event tagging using FC Python video coder
- **Events Tracked**: Shots, passes, carries, crosses, goals, assists
- **Sample**: All home matches from first half of 2025/26 season
- **Limitations**: Analysis based on match highlights (acknowledged in report)

## Technical Stack
- Python 3.x
- Pandas (data manipulation)
- Matplotlib & Seaborn (visualization)
- mplsoccer (football-specific plots)
- LaTeX (report generation)

## Key Findings
- 38 shots total, 15.8% conversion rate
- Average shot distance: 15.1 meters
- Central build-up play with wide finishing
- Strong defensive organization (12.5% opponent conversion rate)
